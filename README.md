# Gomantak Gausevak Mahasang

Official website for Goa's largest stray cattle shelter.

## Live Site
https://sarthaksri06.github.io/Gaumantak-Gauseva

## Features
- Responsive design with Tailwind CSS
- Animated stat counters
- Glass-morphism testimonial carousel (8 eminent personalities)
- YouTube video integration
- Donation & adoption CTAs
- Mobile-first navigation

## Tech Stack
- HTML5
- Tailwind CSS
- Vanilla JavaScript
- Font Awesome
- Google Fonts

- Home
